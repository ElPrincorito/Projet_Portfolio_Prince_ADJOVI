Mini-jeu Interactif en JavaScript

Thème : Développement Front-End, Logique JavaScript

Ce mini-jeu interactif en JavaScript illustre l'application pratique des concepts de la programmation orientée objet (POO) côté client. Le jeu met en avant la gestion de l'état, les interactions utilisateur et les animations dynamiques, le tout sans recourir à des bibliothèques externes.

Objectif et Approche

L'objectif de ce projet était de démontrer une maîtrise des structures de contrôle, de la manipulation du DOM et de la Programmation Orientée Objet (POO) en JavaScript sans utiliser de bibliothèque externe.

Défis Relevés

  • Gérer l'état complet du jeu (scores, niveaux, vies) de manière cohérente et fiable.
  • Assurer une interface utilisateur (UI) réactive et des animations fluides uniquement avec CSS et JS, optimisant les performances.

Solution Technique

Le jeu est entièrement développé en **JavaScript vanilla**. La logique centrale repose sur des objets JavaScript pour encapsuler les propriétés et les méthodes de chaque élément du jeu (joueur, ennemis, environnement), garantissant un code clair et maintenable. L'interaction avec l'utilisateur est gérée par des écouteurs d'événements bien structurés.

Technologies : HTML5, CSS3, JavaScript (POO).
